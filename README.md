# GST-8A-2A-2B-Reporter
This generates a report on GST 8A/2A/2B excel data uploaded in a streamlit. The Report generated based on Interstate/Intrastate supply and its GST Rate with a Client wise Summary Report as a seperate sheet in it.
Most Important to Note : Take away the Rupee Symbol is the Taxable value, Integrated xax, Central tax, State/UT tax Column & then merge the sub columns under Invoice details (4) and Tax Amount (4) Column as each single column
